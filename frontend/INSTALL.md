# Instalación de dependencias

Para resolver los errores de TypeScript, necesitas instalar las dependencias del proyecto:

```bash
cd frontend
npm install
```

O si prefieres usar pnpm:

```bash
cd frontend
pnpm install
```

Después de instalar las dependencias, los errores de TypeScript deberían desaparecer ya que TypeScript podrá encontrar los tipos de React, Vite y otras librerías.
